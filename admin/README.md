# 博客管理端

## 使用方式

1. 双击 `BlogAdmin.exe`。
2. 管理端会以独立小窗口打开，默认管理 `E:\Blog` 里的 Hexo 博客。
3. 支持文章、书籍/书摘、画廊、分类、日记、电影。
4. 点“保存”才会写回博客源文件，并给被覆盖的文件生成 `.bak` 备份。
5. 点“一键发布”会按顺序执行 `hexo clean`、`hexo generate`、`hexo deploy`。

## 书摘

- 书籍如果有 `notes` 链接，管理端会自动读取对应的 `source/books/notes/.../index.md`。
- 书籍编辑页会在下方显示书摘 Markdown，可直接编辑。
- 保存时会写回 `source/_data/books.yml`，并把书摘内容写回对应书摘文件。

## Markdown 编辑器

- 文章正文和书摘内容提供源码与可视化预览。
- 预览支持 H1-H6、列表、引用、代码块、行内代码、表格、链接等常用 Markdown 样式。
- 公式支持 `$...$` 和 `$$...$$`，会预览积分、分式、根号、上下标、常见希腊字母等。
- 标题会用 `<一级标题>` 到 `<六级标题>` 做层级标识。
